校园网宽带密码破解

发表于 2012-03-02| 更新于: 2017-12-19| 分类于 安全研究， 破解

校园网使用了一个叫做安腾的私有拨号客户端，拨号密码似乎加密了，看了下 MSDN 文档，发现拨号的 API 函数是 RasDial，然后载入 ollydbg 然后对这个函数下断点，然后运行拨号客户端拨号，到这个函数的时候程序停止，然后看堆栈信息就可以找到真实的拨号账号密码了，找到真实账号密码就可以到路由器直接拨号。
